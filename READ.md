# Pokédex

Desafio do processo seletivo da empresa Medipreço

## Tecnologias

- React Native
- Expo
- PokéApi

## Dependências

- Yarn
- Expo

## Para rodar o projeto:

- Clone o projeto

```
https://github.com/lucascaldasb/pokedex.git
```

- Instale as dependências

```
$ expo install
```

- rode o projeto

```
$ expo start
```
